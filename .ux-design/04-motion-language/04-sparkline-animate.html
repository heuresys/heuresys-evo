04.C

Use case map

data reveal pattern · max 200ms anti-slop
✅ OK
KPI dashboard cards
Pattern canonical. Header ring 4-col, una volta sola al mount/scroll.
✅ OK
Sparkline trend table column
Mini sparkline per row. Trigger via IntersectionObserver, not on scroll.
✅ OK
Gauge / radial progress
stroke-dasharray sull'arco. 200ms ease-out. Stessa tecnica.
✅ OK
Skill bar / progress fill
width transition 0→target o stroke-dasharray on path.
❌ NO
Dense table cells (numeric)
100+ rows × count-up = freeze CPU. Mostra valore finale subito.
❌ NO
Filter result count update
Update count su filter change deve essere IMMEDIATO. No animation, snap.
🟡 CARE
Bar chart bars grow
OK se <20 bars. Stagger 30-50ms tra bar. Stop a 200ms TOT per chart.
❌ NO
Live updating numbers (websocket)
Real-time data: snap, no count-up. User vuole "verità ora", non animation.

04.E

Anti-pattern banned

data reveal patterns vietati · UX violations
14.011
❌ Slot machine scroll

Numbers scrolling vertically

Pattern casino. Distrae da lettura, suggerisce "incertezza". User vuole valore preciso, non lottery.
14.011
❌ Bouncy number

Scale 1.3 spring loop

Distrazione continua. User non riesce a leggere altri KPI vicini. Off-tone B2B.
14.011
❌ Infinite redraw

Fade-in loop on scroll

Re-anima ogni volta che entri/esci viewport. Crea epilettic flicker. One-shot rule.

// hooks/useCountUp.ts — rAF count-up locale-aware export function useCountUp(target: number, duration = 200) { const [val, setVal] = useState(0); useEffect(() => { const start = performance.now(); let raf: number; const step = (now: number) => { const t = Math.min(1, (now - start) / duration); const eased = 1 - Math.pow(1 - t, 3); // ease-out cubic setVal(target * eased); if (t < 1) raf = requestAnimationFrame(step); }; raf = requestAnimationFrame(step); return () => cancelAnimationFrame(raf); }, [target, duration]); return val; } // components/KPICard.tsx 'use client'; import { useInView } from 'react-intersection-observer'; export function KPICard({ target, points }) { const { ref, inView } = useInView({ threshold: 0.5, triggerOnce: true }); const val = useCountUp(inView ? target : 0); const fmt = new Intl.NumberFormat('it-IT'); return ( <div ref={ref} className={`kpi ${inView ? 'animate' : ''}`}> <div className="num">{fmt.format(Math.round(val))}</div> <Sparkline points={points} /> </div> ); }
