02.C

Use case map

color tokens transition ✅ · layout/size transitions ❌
✅ OK
Theme switch dark↔light
background, color, border-color, box-shadow. Tutti color tokens insieme.
✅ OK
Hover state
card lift, button hover bg, link color. 150ms ease-out.
✅ OK
Focus ring (form)
border-color + box-shadow. 150ms ease-out.
✅ OK
Filter pill toggle
background + color + border. 150ms ease-out.
❌ NO
Layout / grid
Mai transition su grid-template, flex-direction, position. Layout shift = jank.
❌ NO
Width / height
Eccezione: sidebar collapse `width 200ms`. Default mai. Resize causa reflow.
🟡 CARE
Route navigation
Solo via View Transitions API moderna. NO fade overlay che blocca interazione.
❌ NO
Modal enter/exit
Use scale+opacity (200ms), NOT color transition. Modal = focus shift, no token bleed.

02.F

Integration code

CSS · React + next-themes · Tailwind
/* heuresys theme transition — global rule */ :root { --tx-dur: 200ms; --tx-ease: cubic-bezier(0.16, 1, 0.3, 1); } /* Apply to all colored UI elements — NEVER layout */ body, .panel, .btn, .input, .card, .table, .nav-bar, .footer { transition: background var(--tx-dur) var(--tx-ease), color var(--tx-dur) var(--tx-ease), border-color var(--tx-dur) var(--tx-ease), box-shadow var(--tx-dur) var(--tx-ease); } /* Theme switch via attribute */ [data-theme="dark"] { --bg: #0a0d18; --ink: #f4f5f7; ... } [data-theme="light"] { --bg: #f4f5f7; --ink: #0a0d18; ... } @media (prefers-reduced-motion: reduce) { *, *::before, *::after { transition-duration: 0.01ms !important; } }
// app/layout.tsx — using next-themes import { ThemeProvider } from 'next-themes'; import './globals.css'; export default function RootLayout({ children }) { return ( <html suppressHydrationWarning> <body> <ThemeProvider attribute="data-theme" defaultTheme="dark"> {children} </ThemeProvider> </body> </html> ); } // components/ThemeToggle.tsx 'use client'; import { useTheme } from 'next-themes'; export function ThemeToggle() { const { theme, setTheme } = useTheme(); return <button onClick={() => setTheme(theme === 'dark' ? 'light' : 'dark')}>◐</button>; }
