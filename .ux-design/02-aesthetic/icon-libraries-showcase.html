Raccomandazione per Heuresys · 3 candidati shortlisted

Selezione basata su: aderenza estetica brand (mono-line + single accent), copertura dashboard B2B enterprise, ecosistema React/Next.js, license MIT/Apache, no ads/premium tier, manutenzione attiva.

  • 1° Lucide · default in shadcn/ui (stack frontend Heuresys), 1,500+ icone, MIT, tree-shakeable. Aderenza naturale con Tailwind/Vercel ecosystem. Risk: solo line, no fill states.
  • 2° Phosphor · 6 weights stessa famiglia → ottimo per gerarchia density (sidebar = light, KPI = bold, hover = duotone). 9k variants. Più flessibile per Heuresys data-dense + ontology emphasis. Risk: bundle se importi tutti i weights.
  • 3° Tabler Icons · 5,800+ set più ampio, dashboard-first design intent. Coverage enterprise (charts, files, devices, business operations). Risk: meno editorial, più "admin panel" feel.

Decisione finale: pending Enzo. Suggested combo: Lucide per UI core (sidebar nav, table actions, ui controls) + Phosphor duotone per dataviz emphasis (5 capability glyphs custom restano illustrati custom-made).
