Note di lettura

Set 1 (α/β/γ/δ): direzioni più "categoriali" già esplorate, ognuna con singola modalità (dark per α, light per β/γ/δ).

Set 2 (ε/ζ/η/θ): nuove, ispirate da award winners 2025/2026, ognuna con theme toggle light/dark in pari dignità. Trend chiave assorbiti: variable typography sculpturale (CSS Awards), warm palette anti-tech (Centersquare), Swiss pure rigor (Pentagram tradition), generative network (OFF+BRAND × Lando Norris).

Cosa fare ora: naviga le 8 direzioni. Per le NEW, prova il theme toggle in alto a destra (◐ Light/Dark). Quando hai trovato la combinazione che ti tira di più, dimmela. Possiamo anche ibridare elementi tra le 8 (es. "logo θ + tipografia ε + palette ζ") se nessuna ti convince intera.
