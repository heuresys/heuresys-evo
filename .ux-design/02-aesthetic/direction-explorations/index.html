Note di lettura

Set 1 (α/β/γ/δ): direzioni più "categoriali" già esplorate, ognuna con singola modalità (dark per α, light per β/γ/δ).

Set 2 (ε/ζ/η/θ): award-inspired 2025/2026, ognuna con theme toggle light/dark in pari dignità. Trend chiave assorbiti: variable typography sculpturale (CSS Awards), warm palette anti-tech (Centersquare), Swiss pure rigor (Pentagram tradition), generative network (OFF+BRAND × Lando Norris).

Set 3 (ι/κ/λ/μ): nascono dall'output del prompt family-picker.md (Q1 mix scan-heavy prevalente · Q2 prosumer · Q3 trustworthy 60% / courage 40%). Family raccomandata Data-Dense Pro con secondaria Editorial Minimalism. Ogni mockup ha un .critique.md con il prompt 3-designer-debate.md applicato — Architect/Art Director/Pragmatic in 3 round + synthesis + 3 action items concreti.

Set 4 (μ persona variants): 4 esecuzioni alternative del modello μ Data-Dense Temperata che incorporano i 3 punti di vista del critique. μ-architect (tokens visibili + RBAC governance) · μ-art-director (5 capability glyphs ontology + Pentagram editorial) · μ-pragmatic (CTA + KPI big + trial banner) · μ-synthesis (best compromise 40/30/30). Set 4 mostra esplicitamente cosa cambia visivamente dal POV di chi guarda.

Standard logo y-accent (L16): tutti i 16 mockup applicano lo stesso principio — la "y" del wordmark è in var(--accent) theme-aware, weight gap moderato (body 700-800 → y 500-600), italic preferito, letter-spacing naturale. Vedi logo-standard.md.

Cosa fare ora: naviga le 16 direzioni. Toggle theme su tutte. Per Set 3, leggi i 4 critique.md. Per Set 4, confronta come la stessa famiglia (Data-Dense Temperata) si trasforma sotto 3 POV diversi + il loro compromise. Quando hai preferenza per una direction (o una μ-variant), dimmela: chiudiamo D1 e procediamo con Phase 5/6/7/8/9.
