## Known issues

### Application / DevX

- **P3 NEW S2** 2 transitive `postcss <8.5.10` moderate vulnerabilities in `node_modules/next/node_modules/postcss` (XSS via unescaped `</style>`). Not fixable senza downgrade Next 16 → 9 (broken). Build-time-only, non-exploitable in production runtime. Mitigation: monitor Next.js patch releases (track `next@16.x.y` ETA fix).

### Database / Operational

- **P2 OPEN S1** Off-VM backup ora funziona via OCI bucket. Lifecycle policy 30-day pending IAM (P3 NEW S2 below).
- **P3 NEW S2** OCI lifecycle policy 30-day delete blocked by `InsufficientServicePermissions` for `objectstorage-eu-milan-1` service principal. Without it, OCI bucket accumulates indefinitely. Mitigation: rotation manuale via `oci os object delete` se serve, oppure IAM grant via OCI Console.
- **P3 NEW S1** Migration 221 non applicata in nessun ambiente (1 user con plain TOTP secret). Backfill TOTP richiesto. Non bloccante per `.evo` rebuild ma rilevante per ADR-0003 auth.
- **P3 NEW S1** Container `heuresys_evo_enrichment` (sulla VM) status `Up 7 days (unhealthy)` — preesistente, ereditato dal v1. Non rilevante per `.evo` ma nota presente.

### Repo / Filesystem

- **P3 NEW S1** Cancellato per errore il repo v1 su VM con `rm -rf` durante setup `.evo`, recuperato via `git clone` da GitHub. Lezione documentata in memory `feedback_no_rm_rf_without_inspection.md`.
- **P3 NEW S1** Tar pipe da Git Bash su Windows con `cd` ha bug intermittente. Workaround: `tar -C <abs-path>` esplicito.
- **P3 NEW S2** Bash tool (Claude Code CLI) `cd` è persistente cross-call nello stesso shell session. Se lanci background commands con `cd <relpath>`, può fallire se cwd è già lì da invocazione precedente. Workaround: usa path assoluti.
