Alternatives compare

heuresys
★ Recommended
Subtle pulse
4s · ease-in-out · 30↔60px · 0.20↔0.45
Trustworthy 60% / Courage 40%. Alive but calm. Non distrae lettura.
heuresys
Too fast
Fast pulse 1.5s
1.5s · ease-in-out · 25↔50px
Energetic ma overwhelming. Crea fatica visiva, sembra "alarm" non signal.
heuresys
Mechanical
Linear breathing
4s · linear (no easing curve)
Movimento robotico, mancanza di "respiro". Easing curve = umanità.
01.E

Anti-pattern banned

cosa NON fare mai · accessibility + brand violations
heuresys
❌ Strobe blinking

Toggle 200ms hard cut

WCAG violation: rischio epilessia fotosensibile. + signal "broken/error".
heuresys
❌ Rainbow hue cycle

Color cycling glow

Brand violation: viola single-accent rule (`heuresys.DESIGN.md` Section 7). Off-brand teen aesthetic.
heuresys
❌ Bouncy scale

Spring bounce 1s loop

Tone violation: enterprise B2B trustworthy. Bouncy = playful consumer brand (Duolingo), not heuresys.

/* heuresys wordmark glow breathing — canonical */ @keyframes glow-breathe { 0%, 100% { filter: drop-shadow(0 0 30px rgba(168, 85, 247, 0.20)); } 50% { filter: drop-shadow(0 0 60px rgba(168, 85, 247, 0.45)); } } .wordmark-hero { font-family: 'Exo 2', sans-serif; font-weight: 700; color: var(--brand-blue); animation: glow-breathe 4s cubic-bezier(0.45, 0, 0.55, 1) infinite; } .wordmark-hero .y { color: var(--accent); font-weight: 500; } /* Accessibility — respect prefers-reduced-motion */ @media (prefers-reduced-motion: reduce) { .wordmark-hero { animation: none !important; } }
